팀프로젝트 주제: Weather Go (웨더고)

팀프로젝트 설명:

날씨와 기후는 밀접한 관계가 있습니다. (선행연구 참조)
Weather Go는 이러한 관계를 통해 날씨에 따른 관광지를 추천해줌으로써
관광수요자(잠재 소비자 포함)에게 날씨에 따라 관광지를 선택하는데 에 있어서
불필요한 검색 없이, 지역만 검색하면 오늘 같은 날씨에
어디를 가면 좋을지 고민하는데에 있어서 관광지를 추천해줌으로써,
도움을 제공하는 서비스입니다.
